# Career+

## Description

**Career+** is a digital platform prototype designed to support students in finding, applying for, and managing international Erasmus+ internships. It serves as a unified space connecting students, employers (companies and academic institutions), and universities â€” simplifying administrative processes, sharing knowledge, and building a European career network centered around the University of Zagreb.

This is a **frontend-only mockup** built to demonstrate the concept and user interface of the platform.

## How to run

### Prerequisites

- **Node.js 22.19.0+** (or latest LTS)
- **pnpm** (recommended) or npm/yarn
  ```bash
  npm install -g pnpm
  ```

### Install Dependencies

```bash
# Install main dependencies
pnpm install
```

### Run Development Server

```bash
pnpm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.
